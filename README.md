## Installation

```bash

```

## Running the App

```bash
npm start
```

Then open your browser to `http://localhost:3000`

## Usage

1. Start typing in the editor
2. Your note is automatically saved
3. Toggle between "Raw" and "Markdown" view
4. Click "Share" to get a shareable link
5. Click "New Note" to create a fresh note
6. Click "Delete" to remove the current note
